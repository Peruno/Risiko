# Risiko Simulator

Eine Flutter App zur Berechnung der exakten Würfelwahrscheinlichkeiten und Simulieren von Ergebnissen beim Brettspiel
Risiko. Sie benutzt die "Michelson'sche Verzögerungstaktik", eine Bezeichnung für die speziellen Würfelregeln, die wir
in meinem Freundeskreis verwenden. Siehe [docs/latex/output/main.pdf](docs/latex/output/main.pdf)

## Entwicklung

Formatierung in dieser Codebase:`dart format --line-length=120 .`
